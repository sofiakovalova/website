JavaScript — мова програмування вебу

JavaScript — це мова програмування, яка дозволяє створювати інтерактивність на веб-сторінках: обробку подій, анімації, динамічний контент.

Приклад коду:

        document.getElementById("demo").innerHTML = "Привіт, світ!";

JavaScript підтримує роботу з API, сервером (через AJAX чи Fetch), а також використовується у популярних фреймворках React, Angular і Vue.

JavaScript додає динаміку. Почни з простого: роби кнопки інтерактивними.
