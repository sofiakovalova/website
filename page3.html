CSS — каскадні таблиці стилів

CSS (Cascading Style Sheets) відповідає за зовнішній вигляд веб-сторінок. За допомогою CSS можна задавати кольори, шрифти, відступи, розташування блоків та створювати адаптивний дизайн.

Приклад стилю для абзацу: 

        p {
            color: blue;
            font-size: 18px;
          }

Сьогодні широко використовуються CSS-фреймворки, такі як Bootstrap чи Tailwind, які прискорюють верстку сайтів.

CSS допомагає створювати стиль і красу сайту. Використовуй класи для повторюваних елементів!
